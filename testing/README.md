### testing

- sphinx-tribes (local auth plugin setup)
- sphinx-auth
- sphinx-meme (host=localhost:5000)
- btcd
- 4x simnet LND (alice, bob, casey, proxy)
- sphinx-proxy (bitcoin.simnet)
- 4x relay (alice, bob, casey, proxy)
